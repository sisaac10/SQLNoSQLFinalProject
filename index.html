I590 SQL NoSQL

Indiana University


This website was created to present my work related to my Final Project for I590 SQL NoSQL class taught by Professor Ying Ding at Indiana University. The presentation of this work is strictly for academic purposes


This project is based on the evaluation of data related to the three different datasets:

  • 2016 World Happiness Report
  • World Education Expenditure
  • Times World University Ranking

I created two different NoSQL databases, mongodb and XML, to store and query the data.
